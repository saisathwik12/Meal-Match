Key Features

Ingredient-Based Recipe Search

This is the core functionality. Users can input a list of ingredients they already have, and the app utilizes the MealDB API to find recipes that incorporate those specific ingredients. This directly addresses the user's need to find meals based on available resources, minimizing waste and maximizing convenience.

Detailed Meal Information Display

Beyond just showing meal names and images, the app provides comprehensive information about each recipe. This includes a detailed ingredients list with quantities, step-by-step cooking instructions, and links to external resources like YouTube tutorials or the full MealDB recipe. This feature empowers users to confidently prepare the suggested meals.

Dynamic and Responsive User Interface

The app features a user-friendly interface with meal cards displayed in a responsive grid layout. This ensures that the app is accessible and easy to use on various devices, from mobile phones to desktop computers. The dynamic display of ingredients, up to 20 items, ensures that the user recieves all needed information.
